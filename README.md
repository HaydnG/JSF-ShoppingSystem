# JSF Shopping System
A JSF web application using the command interface design pattern.

## Features

* Login, Registration
* Add/Remove from cart
* Create/View Order
* Add/Remove users
* Add/Remove Products
* View Products

## What I Learned

* Use of java and jsf to make a web applicaton
* Command Design pattern
* Use of ajax to make pages interactive
