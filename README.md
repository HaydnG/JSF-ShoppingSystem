# JSF Shopping System
A JSF web application using the command interface design pattern.

## Features

* Login, Registration
* Add/Remove from cart
* Create/Place/View Order
* Order History
* Add/Remove users
* Add/Remove Products
* View Products
* Full site management from an Admin account
